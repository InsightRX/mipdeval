# Options for controlling MAP Bayesian fit

Options for controlling MAP Bayesian fit

## Usage

``` r
fit_options(..., reltol = 1e-05)
```

## Arguments

- ...:

  These dots are reserved for future extensibility and must be empty.

- reltol:

  Relative convergence tolerance. `reltol = 1e-04` will perform a
  slightly coarser but more stable fit, which can be useful in some
  case.

## Value

A list.
